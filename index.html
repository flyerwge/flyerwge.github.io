利用Hexo搭一篇属于自己的博客

一、准备工作

1.搭建Node.js环境

下载地址：http://nodejs.cn/download
测试安装：命令行使用node -v 、npm -v，查看显示版本号即成功。

2.安装Hexo博客框架工具

命令行使用:

第一句是安装hexo，第二句是安装hexo部署到git page的deployer。

3.安装Git工具

下载地址：https://git-scm.com/downloads
测试安装：命令行使用git –version,查看显示版本号。

4.注册Github账号

注册地址：https://github.com

二、开始搭建博客

1.新建仓库


其中仓库名最好选择“username+github.io”的形式，使用其他形式会导致博客网站的链接过于冗余。

2.设置Github pages

点击Create Repository之后，随后选择Setting进入设置，找到Github Pages如下：

选择一个主题，此时就可通过username.github.io访问自己最初的博客网站。

3.创建本地博客站点

在合适的文件夹下右键点击“Git Bash Here”（控制台命令）,然后输入hexo init myFirstBlog //生成项目创建自己的第一个博客，也可通过在source/posts文件夹下新建md文件生成博客。

4.测试本地博客站点

在本地博客根目录下使用控制台命令：

打开浏览器，在地址栏输入localhost:4000,就能看见Hexo自带默认主题。

5.同步Github，允许公共访问

找到自己的博客仓库，拷贝地址；

然后修改本地博客根目录下的_config.yml文件，在deploy下的配置添加如下：

其中仓库地址为自己在Github中复制的仓库地址；
最后控制台命令
hexo d //部署到GitHub
以后在写自己的博客时可以将d、g命令合并为
hexo d -g
现在就可以打开自己的博客网站：usename.github.io

三、更换主题

1.选择主题

Hexo主题网站：https://hexo.io/themes/
从中选择一款自己喜欢的主题，点击主题界面可提前阅览，点击主题名称进入主题详情页面，详情页面中有install方法。

2.主题安装

在博客根目录下执行控制台命令：
git clone "url" themes/"themeName"(大多主题中会有安装命令)，其中:
url:主题地址
themename:主题名称

3.主题配置

  1. 主题下载成功后在博客根目录下的themes文件夹中会出现自己所下载的主题；打开博客根目录下的_config.yml文件，将主题配置改为当前选择的主题名称：
  2. 通过命令进入当前主题文件夹：
    cd themes/landfarz
  3. 执行如下代码,将当前更改的信息合并：
    git pull
  4. 最后主题配置完成，回到博客根目录下重新部署整个博客，在博客根目录下执行控制台命令：
    hexo d -g

四、将博客与域名关联

1.购买域名

在阿里云(或者GoDaddy等)购买自己想要的域名；

2.配置CNAME文件

在 \hexo\source 文件夹下创建文件 CNAME （新建记事本文件命名CNAME，无后缀名，然后打开），在在首行添加网站域名，如http://xxxx.com,注意前面没有http://，也没有www,然后上传部署；

3.域名解析

进入DNSPOD注册账户、添加域名，并且修改记录值；
记录值修改：

  1. 先添加一个CNAME，主机记录写@，后面记录值写上你的http://xxxx.github.io
  2. 再添加一个CNAME，主机记录写www，后面记录值也是http://xxxx.github.io
    （如果用A记录，记录值则是自己的GitHub page对应的IP地址）
