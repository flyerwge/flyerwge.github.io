利用Hexo搭一篇属于自己的博客

一、准备工作

1.搭建Node.js环境

下载地址：http://nodejs.cn/download
测试安装：命令行使用node -v 、npm -v，查看显示版本号即成功。

2.安装Hexo博客框架工具

命令行使用:
‘npm install hexo-cli -g’
‘npm install hexo-deployer-git –save’
第一句是安装hexo，第二句是安装hexo部署到git page的deployer。
